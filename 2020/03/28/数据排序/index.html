​ 数据排序

排序（升序，降序）

按照工资升序，找出员工名和薪资。

select ename,sal from emp order by sal;

select ename,sal from emp order by sal asc/desc; asc为升序desc为降序，不指定的话默认升序。

按照工资的降序排列，当工资相同的时候按名字的升序排列

select ename,sal from emp order by sal desc,ename asc;

注意：越靠前的字段越能起到主导作用，只有当前面的字段无法完成排序的时候，才会启用后面的字段。

按字段名进行排序：select * from emp order by ename;

select * from emp order by sal;

找出工作岗位是salesman的员工，并且要求按照薪资降序排列：

select ename,sal,job from emp where job=’salesman’ order by sal desc;
