动态初始化一维数组

​ 动态初始化一维数组

1、动态初始化一维数组，会先在堆内存中分配这个数组，并且数组中每一个元素都采用默认值。

byte,short,int,long 0

float,double 0.0

boolean false

char \u0000

引用

2、什么时候抵用动态初始化，什么时候使用静态初始化？

——无论使动态初始化还是静态初始化，最终的内存分布都是一样的。

——如果在创建数组的时候，知道数组中应该存储什么数据，就采用静态初始化。

——如果在创建数组的时候，无法预测到数组中存储什么数据，只是先开辟空间，则使用动态初始化方式。
