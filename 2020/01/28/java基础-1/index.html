java基础

​ 第一章

标识符：

——在java源程序中只要是程序员有权利自己命名的单词都是标识符。

——在标识符可以标识什么什么元素

​ 类名，方法名，变量名，接口名，常量名。

——标识符的命名规则

​ 一个合法的标识符只能由数字，字母，下划线，美元符号组成，不能有其他符号。

​ 不能以数字开头。

​ 严格区分大小写。

​ 关键字不能做标识符。关键字即java定义过的如public，class。

​ 理论上无长度限制。

——标识符的命名规范

​ 只是一种规范不属于语法，不遵守规范不会报错，但是程序写完别人看不懂。

​ 最好见名知意，如：

​ 遵守驼峰命名方式：

​ SystemService， UserService,CunstomerService等等

​ 类名，接口名：首字母大写，后面每个单词首字母大写。

​ 变量名，方法名：首字母小写，后面每个单词首字母大写。

​ 常量名：全部大写。

​ 第二章

——数据在现实生活中是分门别类的，所以数据在计算机编程语言中也是有类型的称之为【数据类型】。

​ 10，100 属于整数型字面值

​ 3.14 属于浮点型字面值

​ true，false 属于布尔型字面值

​ “abd“，”中国人“ 属于字符串型字面值

​ “A”，“人” 属于字符型字面值

——注意：java语言中所有字符串型字面值必须使用双引号括起来，半角双引号

​ java语言中所有字符型字面值必须用单引号括起来，半角单引号

​ 第三章

——变量：变量本质上来说是内存中的一块空间，这块空间有“数据类型”，”名字“，”字面值“。

​ 变量包含三部分：数据类型，名称，字面值【数据】。

​ 变量是内存中存储数据的最基本单元。

——数据类型的作用：

​ 不同的数据有不同的类型，不同的数据类型底层会分配不同大小的空间。

​ 数据类型是指导程序在运行阶段该分配多大的内存空间。

——声明/定义变量的语法格式：

​ 数据类型 变量名；；

​ 数据类型： 如 int 整数型 double float等。

​ 变量名：只要是合法的标识符就行，规范中要求：首字母小写，后面每个单词首字母大写。

​ 例如：int i；

                int age；

​ int length；

​ int size；

​ int num；

​ 其中int是数据类型，i，age，length，size，num是变量名。

——变量声明之后如何赋值

​ 语法格式：变量名=字面值

​ 要求：字面值的数据类型必须和变量的数据类型一直。

​ =等号是一个运算符，叫做赋值运算符，赋值运算符先运算等号右边的表达式，表达式执行之后的结果赋值给左边的变量。

——声明和赋值可以放到一起完成

​ int i=10

——赋值后可以重新赋值，变量的值可以变化；

——通常访问变量的两种形式

​ 第一种：读取变量中保存的具体数据 get/获取

​ 第二种：修改变量中保存的具体数据 set/设置

—— i = 20 ；//set

​ System.out.println(i) ; // get

——变量在一行上可以声明多个

​ int a，b，c；

——在java方法体当中的代码，是遵守自上而下的顺序逐行执行。

​ 特点：前一行的代码必须完整结束之后下一行才能执行。

——在同一个“作用域”当中，变量名不能重复。

——什么是作用域：变量的作用域，其实描述的就是变量的有效范围。在什么范围内是可以被访问的，只要出了这个范围该变量就无法访问。

——变量的作用域只要记住一句话：出了大括号就不认识了
